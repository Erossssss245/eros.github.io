Changelog

Full release history for the mod. Versions are listed newest first.

🆕 v1.1.0

  • Added Crystal Bow — a new ranged weapon that uses Luminite Dust as ammo
  • Added Verdant Forest biome with new Verdant Log wood type and all plank variants
  • Added Glowbeetle passive mob that spawns in Crystal Caverns
  • Changed Luminite Crystal ore now generates in larger veins (max vein size increased from 4 to 8)
  • Changed The Archon boss health reduced by 10% in Normal difficulty
  • Fixed Shadow Walker mob was not dropping Shadow Essence consistently
  • Fixed Crystal Cavern biome could occasionally overwrite Strongholds, preventing End Portal generation

🐛 v1.0.1

  • Fixed Mod Workbench crafting UI would sometimes appear blank on first open
  • Fixed Shadow Cloak invisibility effect was persisting through death
  • Fixed Ancient Ruins structure sometimes generated inside mountains, making it inaccessible
  • Fixed Config file not generating correctly on first launch on some systems
  • Changed Shadow Lantern now correctly emits a light level of 8 (was 12)

🎉 v1.0.0 — Initial Release

🎉 First Public Release This is the first stable release of the mod. Thank you to everyone who tested during beta!
  • Added Luminite Crystal ore, Luminite Dust and Crystal Block
  • Added Shadow Essence material and all dark-tier recipes
  • Added Ancient Stone and Ancient Stone Brick decorative blocks
  • Added Mod Workbench block with custom crafting UI
  • Added Luminite Sword, Luminite Pickaxe, Shadow Dagger, Verdant Axe
  • Added Archon Shield armor piece
  • Added Shadow Walker and Stone Sentinel hostile mobs
  • Added Crystal Golem tameable passive mob
  • Added The Archon end-game boss, summoned with the Archon Rune
  • Added Crystal Caverns deep cave biome
  • Added Ancient Ruins surface structure with loot chests
  • Added Shadow Wastes hostile overworld biome
  • Added Luminite Lens, Shadow Cloak, Crystal Compass and Verdant Scroll items
  • Added Config file with world generation and difficulty options

🧪 v0.9.0 (Beta)

Internal beta release for testing. Not publicly available.

  • Added Initial implementation of Luminite Crystal ore generation
  • Added Prototype Mod Workbench block
  • Added Early version of Shadow Walker mob AI
  • Added Placeholder Crystal Caverns biome
