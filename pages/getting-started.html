Getting Started

Welcome to the mod! This page walks you through everything you need to know to get up and running, from installation to your first crafting recipes.

⬇️ Installation

The mod supports both Forge and Fabric mod loaders. Choose the version that matches your setup.

Forge

  1. Download and install Minecraft Forge for the correct MC version.
  2. Download the latest mod .jar file from the releases page.
  3. Place the .jar file inside your .minecraft/mods/ folder.
  4. Launch Minecraft using the Forge profile.

Fabric

  1. Install Fabric Loader and Fabric API.
  2. Download the Fabric version of the mod .jar.
  3. Place both the mod and Fabric API .jar files in .minecraft/mods/.
  4. Launch Minecraft using the Fabric profile.
💡 Tip It is recommended to use a mod manager like CurseForge App or Modrinth App to keep your mods and loader up to date automatically.

🚀 First Steps

Once you have the mod installed, load up a new or existing world. Here is what to do first:

  • Open your inventory to see the new item categories in JEI / REI.
  • Look for new ore types underground when mining below Y=30.
  • Craft a Mod Workbench to unlock advanced crafting recipes.
  • Explore the surface for new structures that may contain loot.
ℹ️ Note Some features are disabled by default in the config file. Check the Configuration section to enable them.

🗺️ World Generation

The mod adds several new biomes and structures to your Minecraft world:

  • Crystal Caverns — deep underground cave biomes filled with glowing crystals.
  • Ancient Ruins — rare surface structures containing mod-specific loot chests.
  • Verdant Forest — a lush forest biome with unique wood types and plant life.
  • Shadow Wastes — a dark overworld biome that spawns mod-specific hostile mobs.
⚠️ Warning World generation changes only take effect in newly generated chunks. Existing chunks in an old world will not be updated.

⚔️ Mobs & Bosses

Several new entities are added by the mod:

Passive Mobs

  • Glowbeetle — a passive insect that drops Luminite Dust used for crafting.
  • Crystal Golem — a friendly golem found in Crystal Caverns that can be tamed.

Hostile Mobs

  • Shadow Walker — spawns in dark biomes and drops Shadow Essence.
  • Stone Sentinel — a ranged attacker found guarding Ancient Ruins.

Bosses

  • The Archon — the main boss of the mod. Summoned using the Archon Rune. Drops the Core Shard needed for end-game crafting.

⚙️ Configuration

A configuration file is generated at .minecraft/config/modname-common.toml on first launch. Key options:

# Enable or disable world gen features
enableCrystalCaverns = true
enableAncientRuins   = true

# Boss difficulty multiplier (0.5 - 2.0)
bossDifficultyMultiplier = 1.0

# Allow mod mobs to spawn in vanilla biomes
spawnInVanillaBiomes = false
